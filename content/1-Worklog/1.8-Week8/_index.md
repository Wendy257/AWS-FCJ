### Week 8 Objectives:

* **Containerization:** Understand Docker fundamentals and containerizing applications.
* **Container Orchestration:** Master Amazon ECS (Elastic Container Service) to manage containerized workloads.
* **Serverless Containers:** Utilize AWS Fargate to run containers without managing servers.
* **Registry Management:** Store and manage container images using Amazon ECR (Elastic Container Registry).
